## TOOL 5: B-Roll Ready Specification Template

### *Define your visual cue conventions. Paste into your Layer 3 prompt or save as a reusable block.*

---

### My B-Roll Conventions

| Element | My Specification |
|---|---|
| B-roll frequency | One visual change every [90 / 120 / 150] seconds of spoken content |
| B-roll specificity level | [e.g., "Must include: brand names, specific UI elements, exact colors, specific objects. No 'show a computer.' Yes 'close-up of Notion sidebar with 14 nested pages, cursor hovering over 'Q3 Planning.'"] |
| Reaction beat frequency | After [every personal confession / every absurd statistic / every contrarian claim] |
| Text on screen usage | For [key statistics / framework names / step numbers / memorable one-liners]. Max [3–4] per script. |
| Visual style preference | [e.g., "Screen recordings for tech content. Handwritten-style text for personal content. Clean data visualizations for educational content."] |
| What I NEVER want visually | [e.g., "Generic stock footage of people typing. Motivational quote cards. Slow-motion walking shots."] |

---

### B-Roll Description Quality Check

Before accepting a B-roll token, verify:

| Check | Pass? |
|---|---|
| Is the visual SPECIFIC? (brand, object, color, action — not generic) | ☐ |
| Does it reinforce or add context to the preceding sentence? | ☐ |
| Is it different from the previous B-roll? (no repeated visuals) | ☐ |
| Could an editor find/create this from the description alone? | ☐ |
| Is it placed at a natural break (not mid-sentence)? | ☐ |

---

### Example B-Roll Descriptions (Good vs. Bad)

| Bad (Generic) | Good (Specific) |
|---|---|
| "Show a person looking frustrated at a computer" | "Over-the-shoulder shot: my actual desk at 11:47 p.m., three empty coffee mugs, Cursor IDE open with a single blinking error on line 247" |
| "Show a productivity app" | "Screen recording: Notion dashboard with 14 database views, the 'Daily Tasks' board showing 23 unchecked items, cursor hovering over the 'Archive All' button" |
| "Show a graph going up" | "Animated line chart: 'Hours spent in meetings' climbing from 4/week (Jan) to 19/week (June), with a red circle around the June spike" |
| "Show someone exercising" | "GoPro chest-mount: my actual running shoes hitting pavement at 6:02 a.m., phone strapped to arm showing Strava at 0.3 miles, breathing visible in cold air" |
