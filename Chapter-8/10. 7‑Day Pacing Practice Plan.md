## 10. 7‑Day Pacing Practice Plan

**Day 1 – Learn the Tokens**  
- [ ] Print the Pacing Token Reference Card.  
- [ ] Read through each token category and practice using them in sample sentences from an old script.  
- [ ] Record yourself reading a short paragraph with and without tokens. Note the difference.

**Day 2 – Build Your First Rhythm Grid**  
- [ ] Pick the framework you use most (PAS or Hero’s Journey).  
- [ ] Use the Rhythm Grid Builder Worksheet to map beats to intensity and tempo.  
- [ ] Save the grid as a reusable template.

**Day 3 – Retrofit an Old Script**  
- [ ] Take a flat, previously recorded script.  
- [ ] Run the Standalone Pacing Retrofit Prompt on it.  
- [ ] Read the retrofitted version aloud. Mark any tokens that feel off and adjust manually.

**Day 4 – Integrate into Layer 3**  
- [ ] Open your Layered Prompting Workspace.  
- [ ] Upgrade your Layer 3 prompt with the Full Pacing Injection Prompt (with your tempo map and the B‑Roll Ready insert).  
- [ ] Add the pacing protection clause to your Layer 4 prompt.

**Day 5 – Full Run with Pacing**  
- [ ] Create a new script from scratch, using your upgraded chain (Layers 1–4).  
- [ ] Record a voice test. Listen back and note the rhythmic feel.  
- [ ] If anything feels off, adjust the tempo map and regenerate Layer 3 only.

**Day 6 – Build a Niche‑Specific Pacing Signature**  
- [ ] Complete the Pacing Signature Builder for your primary niche.  
- [ ] Add a pacing addendum to your Voice Profile.  
- [ ] Test the signature by generating a script without a detailed tempo map—only the signature and token list.

**Day 7 – Reflect and Share**  
- [ ] Compare the retention graph of a video with engineered pacing to one without.  
- [ ] Write a short note: “The biggest rhythmic shift in my scripts now is ________.”  
- [ ] Teach the concept of pacing tokens to a creator friend in a 2‑minute voice note.
