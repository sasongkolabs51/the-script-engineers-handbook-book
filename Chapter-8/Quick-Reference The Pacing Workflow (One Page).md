## Quick-Reference: The Pacing Workflow (One Page)

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   PACING AS CODE — THE WORKFLOW                                 │
│                                                                 │
│   1. GRID (2 min)                                              │
│      → Map emotional intensity per beat (Tool 2 or 3)          │
│      → Translate to tempo: Fast at peaks, Slow at valleys     │
│      → Mark where B-roll goes (every 90–120 sec)              │
│                                                                 │
│   2. INJECT (5 min)                                            │
│      → Run Layer 3 prompt (Tool 4)                             │
│      → Token vocabulary + tempo map + B-Roll Ready             │
│      → AI produces pre-scored script                           │
│                                                                 │
│   3. PERFORM (3 min)                                           │
│      → Read the script ALOUD                                   │
│      → Follow the tokens. Breathe where they say.             │
│      → Does it feel like a performance or a reading?           │
│                                                                 │
│   4. AUDIT (2 min)                                             │
│      → Run the Read-Aloud Pacing Audit (Tool 8)               │
│      → Check: variation, peaks, valleys, breathing, B-roll    │
│      → If off: adjust tempo map, re-run Layer 3               │
│                                                                 │
│   ─────────────────────────────────────────────────────────     │
│   THE RULE: Tokens mark INFLECTION POINTS.                    │
│   Not every sentence. Transitions. Revelations. Peaks.         │
│   The unmarked sentences ARE the rhythm.                       │
│   Silence is a choice. So is speed. So is stillness.          │
│   ─────────────────────────────────────────────────────────     │
│                                                                 │
│   15 TOKENS:                                                   │
│   Pause: [0.5s] [1.5s] [3s]                                   │
│   Speed: [Fast energy] [Slow down] [Build speed]              │
│   Emphasis: [Whisper] [Punch] [Slow emphasis]                 │
│   Visual: [B-roll] [Reaction beat] [Text on screen]           │
│   Transition: [Hard cut] [Fade out energy] [Beat drop]        │
│                                                                 │
│   TARGET: 12 min total pacing work per script.                 │
│   (Down from 45 min of post-production choreography.)          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
