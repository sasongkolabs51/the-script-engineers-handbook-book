## 4. B‑Roll Ready Prompt Insert

Attach this block to any pacing prompt to enforce rich, specific visual cues.

```
B‑ROLL READY INSTRUCTION:
- Insert a [B‑roll moment: describe] every 90‑120 seconds of spoken content.
- Each description must be concrete enough for an editor or stock footage search: name the interface element, object, setting, or action.
- Do not repeat visuals. Vary between screen recordings, physical environments, props, and text overlays.
- After every personal confession or vulnerable statement, add a [Reaction beat] where the presenter looks at the camera silently for 1‑2 seconds.
- For any statistic or key takeaway, add a [Text on screen: “exact quote”] immediately after it.
```
