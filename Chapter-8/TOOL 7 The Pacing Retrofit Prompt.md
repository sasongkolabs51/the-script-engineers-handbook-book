## TOOL 7: The Pacing Retrofit Prompt

### *Add pacing to an existing script. Feel the difference instantly. Use this on an old script to experience the transformation.*

---

```
═══ PACING RETROFIT ═══

Take the following script and inject pacing tokens, rhythm variation,
and visual production cues. Transform it from a readable document
into a performable script.

[PASTE YOUR EXISTING SCRIPT HERE — ideally one that felt "flat"
when you recorded it]

═══ INSTRUCTIONS ═══

1. Read the script and identify its emotional arc. Where does
   tension rise? Where does it peak? Where does it release?
   Where does it settle?

2. Insert pacing tokens at every structural inflection point:
   - [Pause] before revelations and after key statements
   - [Fast energy] in high-urgency sections
   - [Slow down] in reflective/vulnerable sections
   - [Build speed] leading into peaks
   - [Whisper] for confessions or secrets
   - [Punch] on the single most important word in a key sentence
   - [Beat drop] after the longest pause, signaling new energy

3. Insert visual tokens:
   - [B-roll: specific description] every 90–120 seconds
   - [Reaction beat] after personal admissions or absurd moments
   - [Text on screen: "quote"] for key statistics or one-liners

4. Vary sentence length to match the emotional tempo:
   - In [Fast energy] sections: sentences under 10 words
   - In [Slow down] sections: sentences of 18–25 words
   - At peaks: one fragment. Like this. Then the next thought.

5. Do NOT change the content, argument, or vocabulary. Only add
   rhythm, pauses, and visual architecture.

═══ TOKEN VOCABULARY ═══
[List all 15 tokens from Tool 1]

═══ OUTPUT ═══
Return the full script with inline tokens. After the script, add:
- "Before/After Comparison": note the 3 biggest rhythmic changes
  you made and why.
- "Read-Aloud Guide": 2–3 sentences on how to perform this script
  (where to breathe, where to push energy, where to let silence do
  the work).
```

---

### The Retrofit Comparison Exercise

After running the retrofit, do this:

1. **Read the ORIGINAL script aloud.** Time it. Note where you naturally wanted to pause but couldn't (no marker). Note where you rushed.

2. **Read the RETROFITTED script aloud.** Time it. Note how the tokens guide your breath and energy.

3. **Compare:**

| Metric | Original | Retrofitted |
|---|---|---|
| Total reading time | ___ sec | ___ sec |
| Moments I naturally paused (without markers) | ___ | ___ |
| Moments I felt rushed | ___ | ___ |
| Moments I felt the emotional weight land | ___ | ___ |
| B-roll moments I'd need to figure out in editing | ___ | ___ |

**The difference in editing time this would save me:** approximately _________ minutes.
