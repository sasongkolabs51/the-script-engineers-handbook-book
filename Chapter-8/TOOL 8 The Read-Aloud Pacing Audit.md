### The Rhythm Check

| Check | Question | Pass? | If No — Where? |
|---|---|---|---|
| **Breathing** | Are there natural places to breathe? (Pauses at the end of thoughts, not mid-sentence?) | ☐ | |
| **Variation** | Does the tempo CHANGE across the script? (Not all one speed?) | ☐ | |
| **Peaks** | Is there a clear moment of maximum energy/intensity? (The climax?) | ☐ | |
| **Valleys** | Is there a clear moment of minimum energy? (The reflective beat?) | ☐ | |
| **Landing** | Does the ending feel like a landing, not a stop? (Gentle decrescendo?) | ☐ | |
| **Pauses** | Do the pauses feel EARNED? (After something significant, not random?) | ☐ | |
| **Speed shifts** | Do [Fast energy] sections actually feel faster when I read them? | ☐ | |
| **B-roll** | Are visual breaks at natural moments? (Not interrupting a thought?) | ☐ | |
| **Token density** | Are there stretches of 3+ sentences with NO tokens? (Good—normal rhythm.) | ☐ | |
| **Over-tokenization** | Are there stretches where EVERY sentence has a token? (Bad—choppy.) | ☐ | |

### The "Perform, Don't Read" Test

After reading aloud, ask:

> "Did I just READ a script, or did I PERFORM a piece?"

☐ Perform — the tokens guided my energy naturally. I forgot I was reading.
☐ Read — I was conscious of the words. The tokens felt like obstacles.
☐ Mixed — some sections felt performed, others felt read.
