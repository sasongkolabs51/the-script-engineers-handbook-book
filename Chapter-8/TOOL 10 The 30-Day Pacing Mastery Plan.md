## TOOL 10: The 30-Day Pacing Mastery Plan

### *Build rhythmic instinct through deliberate practice. From token placement to full performance scoring.*

---

### Week 1: Token Fluency (Days 1–7)

| Day | Action | Time | Done? |
|---|---|---|---|
| 1 | Study the Pacing Token Reference (Tool 1). Read every example aloud. Feel the difference each token makes. | 15 min | ☐ |
| 2 | Take an old script (one that felt flat). Run the Pacing Retrofit (Tool 7). Read before/after aloud. | 25 min | ☐ |
| 3 | Take a DIFFERENT old script. Retrofit it. This time, choose the token placements yourself before running the prompt. Compare your choices to the AI's. | 25 min | ☐ |
| 4 | Write a 200-word paragraph. Add tokens manually (no AI). Read it aloud. Adjust. Read again. Feel where tokens help vs. hinder. | 20 min | ☐ |
| 5 | Write another 200-word paragraph. This time, read it aloud FIRST, mark where you naturally pause/speed up/slow down. THEN add tokens to match your natural rhythm. | 20 min | ☐ |
| 6 | Run a full Layer 3 prompt (Tool 4) on a current script. Use the token vocabulary but NO tempo map yet. See where the AI places tokens intuitively. | 20 min | ☐ |
| 7 | **Reflection:** Which tokens feel most natural to me? Which do I overuse? Which do I avoid? | 10 min | ☐ |

---

### Week 2: Rhythm Gridding (Days 8–14)

| Day | Action | Time | Done? |
|---|---|---|---|
| 8 | Build your first Rhythm Grid (Tool 2) for a PAS script. Plot the emotional arc. Translate to tempo instructions. | 20 min | ☐ |
| 9 | Run the full Layer 3 prompt WITH your tempo map. Read aloud. Compare to Day 6 (no tempo map). Feel the difference. | 25 min | ☐ |
| 10 | Build a Rhythm Grid for a Hero's Journey script. Use the pre-built grid (Tool 3) as a starting point. Adjust intensities for your topic. | 20 min | ☐ |
| 11 | Run Layer 3 with the Hero's Journey tempo map. Read aloud. Does the Ordeal (Beat 5) feel like a peak? Does the Return (Beat 7) feel like a landing? | 25 min | ☐ |
| 12 | Build a Rhythm Grid for a Hook-Story-Offer script. Run Layer 3. Read aloud. | 25 min | ☐ |
| 13 | Compare all three rhythm grids. Notice: the SHAPE differs per framework. PAS peaks at Agitation. Hero's Journey peaks at Ordeal. HSO peaks at the Hook. | 15 min | ☐ |
| 14 | **Reflection:** Save your three completed rhythm grids as reusable templates. Label them: "PAS Rhythm Grid v1.0," etc. | 10 min | ☐ |

---

### Week 3: B-Roll + Integration (Days 15–21)

| Day | Action | Time | Done? |
|---|---|---|---|
| 15 | Run a full Layered Prompting chain (Layers 1–4) with the upgraded Layer 3 (tokens + tempo map + B-Roll). Time yourself. | 55 min | ☐ |
| 16 | Read the final script aloud. Run the Read-Aloud Pacing Audit (Tool 8). Score it. | 15 min | ☐ |
| 17 | Record yourself performing the script. Listen back. Mark moments where the rhythm felt unnatural. | 20 min | ☐ |
| 18 | Adjust the tempo map for the problematic sections. Re-run Layer 3 + Layer 4 for those sections only. | 20 min | ☐ |
| 19 | Run a second full chain on a different topic. This time, build the rhythm grid BEFORE running Layer 3. Notice how much faster the process is. | 50 min | ☐ |
| 20 | Evaluate the B-roll descriptions. Are they specific enough? Could an editor use them directly? Rewrite any that are generic. | 15 min | ☐ |
| 21 | **Reflection:** How much editing time did pacing tokens save me? Compare to a pre-Chapter-8 script. | 10 min | ☐ |

---

### Week 4: Performance + Refinement (Days 22–30)

| Day | Action | Time | Done? |
|---|---|---|---|
| 22–25 | Produce 3–4 scripts using the full system (rhythm grid + tokens + B-Roll + voice). Each time, read aloud and perform. Track your Pacing Audit scores. | 50 min each | ☐ |
| 26 | Create your "Pacing Signature" addendum: a short paragraph describing YOUR default rhythmic tendencies. (e.g., "I accelerate through arguments and decelerate sharply into insights. I pause before punchlines, not after. My B-roll is screen-recording heavy.") Add to your Voice Profile. | 20 min | ☐ |
| 27 | Test your Pacing Signature on a topic outside your usual niche. Does the rhythm hold? Adjust if needed. | 30 min | ☐ |
| 28 | Run a "blind test": produce a script with NO pacing tokens (old method). Then produce the same script WITH the full system. Read both aloud. Record both. Listen. | 40 min | ☐ |
| 29 | Update your rhythm grid templates based on what you've learned. Refine token density. Adjust B-roll frequency. Version them: v1.1. | 20 min | ☐ |
| 30 | **Final evaluation:** Compare your Day 2 retrofit to your Day 25 full-chain script. Rate both on "performability" (1–10). How much has changed? | 15 min | ☐ |

---

### The Ongoing Practice (After Day 30)

> **Every script:** Build the rhythm grid (2 min) → Run Layer 3 with tokens + tempo map + B-Roll (5 min) → Read aloud (3 min) → Adjust if needed (2 min). Total pacing time: ~12 minutes. Down from 45 minutes of post-production choreography.
>
> **Every recording:** Follow the tokens. Don't override them unless your instinct says a pause should be longer or shorter. The tokens are the score. You're the performer. Trust the score.
>
> **Monthly:** Re-read your rhythm grid templates. Are they still accurate? Has your natural pacing evolved? Update.
>
> **The goal:** Zero post-production pacing work. The script arrives pre-scored. You perform. You don't choreograph.
