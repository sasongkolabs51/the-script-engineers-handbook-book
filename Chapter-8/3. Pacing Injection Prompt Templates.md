```

3. Insert [B‑roll moment: describe] at logical visual breaks (every 90‑120 seconds). Be specific about what the viewer should see.
4. Maintain all original wording and structure; only add pacing tokens and adjust sentence rhythm if needed.
5. Output the full script with tokens inline.

```
