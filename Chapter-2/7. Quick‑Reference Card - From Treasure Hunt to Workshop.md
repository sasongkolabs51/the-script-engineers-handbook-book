## 7. Quick‑Reference Card: From Treasure Hunt to Workshop

*Keep this nearby until the new metaphor feels natural.*

| **Old Mindset (Treasure Hunt)** | **New Mindset (Workshop)** |
|----------------------------------|-----------------------------|
| “I need to find the perfect template.” | “I need to understand the principle behind what makes any template work.” |
| Saving a prompt feels like progress. | Testing a principle and observing the result feels like progress. |
| My value is measured by the size of my swipe file. | My value is measured by the number of principles I can apply on command. |
| When a script fails, I look for another template. | When a script fails, I debug the structural command that was missing. |
| I copy other creators’ words. | I extract the mechanism and rebuild it for my own voice and niche. |
| My growth plateaus because I never go deep on one thing. | My growth compounds because I run deliberate, iterative experiments. |
