```
┌─────────────────────────────────────────────────┐
│                                                 │
│   BEFORE I SAVE ANYTHING, I ANSWER:             │
│                                                 │
│   1. What structural JOB does this do?          │
│   2. What's the PRINCIPLE in one sentence?      │
│   3. Can I rewrite this as a structural         │
│      prompt for MY niche, from memory?          │
│   4. Will I TEST this within 48 hours?          │
│                                                 │
│   Can't answer all four? DON'T SAVE IT.         │

│                                                 │

```
