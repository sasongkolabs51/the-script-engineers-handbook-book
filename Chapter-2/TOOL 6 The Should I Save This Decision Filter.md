## TOOL 6: The "Should I Save This?" Decision Filter

### *A 10-second gate between you and the save button. Run it every single time.*

---

You just found a template, a prompt, a "secret formula," a swipe file entry. Your finger is hovering over Ctrl+S or the bookmark star.

**STOP. Answer these four questions. If you can't answer all four in under 30 seconds, you don't understand it well enough to save it.**

| # | Question | My Answer |
|---|---|---|
| 1 | What **structural job** does this do? (Not "it's a hook." What *mechanism*?) | |
| 2 | What's the **principle** in one sentence, stripped of niche and format? | |
| 3 | Can I write a **structural prompt instruction** that reproduces this mechanism for MY content, right now, without looking at the original? | ☐ Yes ☐ No |
| 4 | Will I **test this principle within 24 hours** in my Prompt Lab? | ☐ Yes ☐ No |

---

**Decision:**

- All four answered confidently → **Extract the principle (Tool 2). Test it today. Then discard the original.**
- Question 3 is "No" → **You don't understand it yet. Don't save it. Study it, break it down, come back when you can rewrite it from memory.**
- Question 4 is "No" → **You're about to add another item to the vault that you'll never use. Close the tab. Walk away. Open your Prompt Lab instead.**

---

### The Emergency Override

If you're in a rush and genuinely can't do the full extraction right now:

☐ Save it to a single folder called **"INBOX — EXTRACT WITHIN 48 HRS"**
☐ Set a calendar reminder for 48 hours from now
☐ If the reminder fires and you haven't extracted it, **delete it without reading it again**

This folder should NEVER contain more than 5 items. If it hits 6, you're back in hoarding mode. Stop. Do the extraction. Now.
