## 2. Layer‑by‑Layer Fill‑in Prompt Templates

Use these exact prompts, replacing only the bracketed parts for each video.

### Layer 1 – Ideation Prompt

```
Topic: [your topic]
Angle: [your specific, contrarian, or personal take]
Target audience: [who you’re speaking to]
Emotional outcome: [what they should feel by the end]
Structure: [chosen narrative framework, e.g., Problem → Hidden Mechanism → Personal Confession → Reframe → Actionable Shift]

Generate a beat sheet with 5–7 distinct beats. Each beat is one sentence describing the content and purpose of that section. Do not write prose yet—only the beat sheet.
```

### Layer 2 – Structural Skeleton Prompt

```
Using the beat sheet below, write a complete script skeleton in raw, functional prose.

Beat sheet:
[PASTE YOUR FINAL BEAT SHEET FROM LAYER 1]

Rules:
- Do NOT add stylistic flourishes, jokes, rhythm markers, or voice personality.
- Focus exclusively on clear logic, smooth transitions, and information flow.
- Include [HOOK] and [CTA] placeholders at the appropriate positions.
- Each beat should become one or two paragraphs of plain, straightforward explanation.

Output the full skeleton.
```

### Layer 3 – Pacing Injection Prompt

```
Inject pacing and delivery markers into the script skeleton below.

Skeleton:
[PASTE YOUR FINAL SKELETON FROM LAYER 2]

Use these tokens where appropriate:
- [Pause 1s]
- [Pause 2s]
- [Fast energy]
- [Slow down]
- [B‑roll moment: describe what should appear on screen]
- [Reaction beat]
- [Whisper emphasis]
- [Laugh break]

Vary sentence lengths to create urgency in high‑stakes sections and breathing room in reflective sections. The final output should feel like a performance script that can be read aloud with natural rhythm and visual cues embedded.

Do NOT change the underlying content or structure—only add pacing markers and adjust sentence rhythm.
```

### Layer 4 – Voice Overlay Prompt

```
Rewrite the following script in my specific voice.

Paced script:
[PASTE YOUR FINAL PACED SCRIPT FROM LAYER 3]

My voice characteristics:
- [e.g., First‑person, conversational but intellectually sharp]
- [e.g., Occasional self‑deprecating humor; dry wit]
- [e.g., Uses sentence fragments and starts sentences with “And” or “But”]
- [e.g., References indie tech culture, productivity experiments, and personal failures]
- [e.g., Never says “in this video we’re going to” or “today I’ll be discussing”]
- [Add any other concrete voice markers you’ve identified]

Keep ALL pacing tokens and structural beats exactly as they are. The goal is to make this sound like me talking to a friend—not like a presenter reading a teleprompter.
```
