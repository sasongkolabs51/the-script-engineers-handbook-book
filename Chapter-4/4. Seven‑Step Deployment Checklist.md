## 4. Seven‑Step Deployment Checklist

*Use this as a real‑time checklist for every script you produce from now on.*

☐ **1. Prepare Your Workspace**  
*Open a new note/document/thread. Title it “Layered Prompting Playbook.” Create four labeled sections (Layer 1–4). Paste the Layer 1 prompt template and fill in the blanks for your video.*

☐ **2. Run Layer 1 — Ideation**  
*Execute the filled‑in prompt. Review beat sheet against Handoff Criterion. Refine and regenerate if needed. Save the final beat sheet.*

☐ **3. Run Layer 2 — Structural Skeleton**  
*Paste beat sheet into the Layer 2 prompt template. Execute. Review skeleton for logic, stakes, transition smoothness. Refine if needed. Save the approved skeleton.*

☐ **4. Run Layer 3 — Pacing Injection**  
*Paste approved skeleton into the Layer 3 prompt template. Execute. Read aloud; check natural rhythm and B‑roll usefulness. Adjust token placement instructions if necessary. Save the paced script.*

☐ **5. Run Layer 4 — Voice Overlay**  
*Paste paced script into the Layer 4 prompt with your best voice description. Execute. Read aloud; ensure it sounds like you. Target specific line‑level tweaks rather than full regeneration. Save the finished script.*

☐ **6. Self‑Assessment Checkpoints**  
*For each layer, before moving on, take 30 seconds to answer the checkpoint question (see Handoff Criterion Card). If any checkpoint fails, spend a few minutes fixing that layer.*

☐ **7. Troubleshoot as Needed**  
*If skeleton reads like a summary: add “At each beat, include a specific, unexpected concrete detail.” If pacing feels random: add emotional stage directions (e.g., “the confession beat should feel slow and vulnerable”). If voice drifts formal: replace abstract descriptors with concrete characteristics you’ll later formalize in Chapter 7.*
