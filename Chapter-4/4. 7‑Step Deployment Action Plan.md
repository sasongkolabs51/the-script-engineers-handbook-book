## 4. 7‑Step Deployment Action Plan

Run this today with a real topic. Target total time: under 60 minutes.

- [ ] **Step 1 – Prepare Workspace.** Duplicate the Layered Prompting Workspace Template. Fill in the topic, angle, audience, emotional outcome, and structure at the top. (3 min)
- [ ] **Step 2 – Layer 1: Ideation.** Paste the Ideation Prompt (filled in) into the AI. Review the beat sheet against the Layer 1 handoff checklist. If it fails, add a refinement instruction (e.g., “Make beat 3 more personal”) and regenerate. Save approved beat sheet. (5–8 min)
- [ ] **Step 3 – Layer 2: Skeleton.** Paste the beat sheet into the Skeleton Prompt. Review for logic, stakes, transitions. If a beat is weak, target that section with a refined prompt and regenerate only that part. Save approved skeleton. (8–12 min)
- [ ] **Step 4 – Layer 3: Pacing.** Paste the skeleton into the Pacing Injection Prompt. Read aloud. If tokens feel random or rhythm is off, add emotional stage directions (e.g., “beats 2–3 should feel urgent; beats 4–5 slow and reflective”) and regenerate. Save paced script. (8–12 min)
- [ ] **Step 5 – Layer 4: Voice.** Paste the paced script into the Voice Overlay Prompt with your current voice description. Read aloud. If any line sounds off, tell the AI to revoice only that section. Save final script. (10–15 min)
- [ ] **Step 6 – Self‑Assessment.** At each layer, spend 30 seconds answering the checkpoint question in the checklist. If any answer is “no,” loop back and fix that layer. (Embedded)
- [ ] **Step 7 – Archive & Reflect.** Save the full playbook entry. Note total time. Write one sentence: “What worked better than my old one‑shot method?” (3 min)
