## TOOL 1: The Layered Prompting Workspace Template

### *Your production floor. Build it once. Reuse it for every script, forever.*

**Instructions:** Create a new document (Notion page, Google Doc, plain text file—whatever you work in). Copy this structure exactly. This is your reusable workspace. For each new script, duplicate the template and fill it in. Over time, you'll build a library of completed workspaces that serve as both portfolio and reference.
