## TOOL 3: Handoff Criterion Checklists

### *Your quality gates. Run these BEFORE advancing to the next layer. 30 seconds each. Non-negotiable.*

---

### Layer 1 Handoff: "Is the Blueprint Solid?"

| # | Criterion | Pass? | If No — What to Fix |
|---|---|---|---|
| 1 | The angle is specific and non-generic (not "tips for productivity" but "why the identity of 'optimized person' is the actual toxin") | ☐ | |
| 2 | The emotional outcome is named and specific (not "informed" but "relieved and slightly angry at how they've been manipulated") | ☐ | |
| 3 | Beats follow a logical sequence where each beat *requires* the previous one (you couldn't reorder them without losing impact) | ☐ | |
| 4 | There's a clear escalation: the stakes or insight deepens as beats progress | ☐ | |
| 5 | A personal element is embedded (your story, your data, your vulnerability) | ☐ | |
| 6 | If I handed this beat sheet to a skilled writer, they could produce a solid script without making further structural decisions | ☐ | |
| 7 | I feel genuine excitement about recording this video based on the outline alone | ☐ | |

**All 7 pass?** ☐ Yes → Proceed to Layer 2. ☐ No → Refine the Layer 1 prompt. Add specificity. Regenerate. Do NOT proceed with a shaky foundation.

---

### Layer 2 Handoff: "Does the Skeleton Stand?"

| # | Criterion | Pass? | If No — What to Fix |
|---|---|---|---|
| 1 | Reading in a monotone voice, the argument holds together logically from start to finish | ☐ | |
| 2 | Stakes are personal and clear — the viewer knows *why this matters to them* within the first two paragraphs | ☐ | |
| 3 | [HOOK] placeholder is positioned correctly (opening) and the beat it sits in has enough material to build a real hook | ☐ | |
| 4 | [CTA] placeholder is positioned correctly (closing) and connects back to the stakes | ☐ | |
| 5 | At least one [TENSION LOOP] is planted and resolved later | ☐ | |
| 6 | Transitions between beats are smooth — no logical jumps or "and then suddenly" moments | ☐ | |
| 7 | No beat feels rushed (underdeveloped) or padded (filler to hit word count) | ☐ | |
| 8 | [SPECIFIC DETAIL NEEDED] markers are present where concrete examples/data should go | ☐ | |
| 9 | The skeleton is deliberately bland — no jokes, no stylized language, no rhythm flourishes | ☐ | |

**All 9 pass?** ☐ Yes → Proceed to Layer 3. ☐ No → Refine the Layer 2 prompt for the specific failing beat. Regenerate that section only.

---

### Layer 3 Handoff: "Does It Breathe?"

| # | Criterion | Pass? | If No — What to Fix |
|---|---|---|---|
| 1 | Reading aloud, I naturally pause where [PAUSE] tokens are placed (they match my breathing instinct) | ☐ | |
| 2 | High-stakes / exciting sections feel faster (shorter sentences, [FAST ENERGY]) | ☐ | |
| 3 | Reflective / vulnerable sections feel slower (longer sentences, [SLOW DOWN]) | ☐ | |
| 4 | [B-ROLL] moments are specific ("show a screen recording of the time-tracking app") not vague ("show relevant footage") | ☐ | |
| 5 | B-roll is placed at logical visual breaks, not mid-sentence or mid-thought | ☐ | |
| 6 | Sentence length variation creates felt rhythm — I can hear the pacing shift | ☐ | |
| 7 | The script feels performable — I could pick it up and deliver it without stumbling | ☐ | |
| 8 | Pacing tokens enhance the emotional arc rather than interrupting it | ☐ | |
| 9 | No section feels monotonously paced (all same speed, all same sentence length) | ☐ | |

**All 9 pass?** ☐ Yes → Proceed to Layer 4. ☐ No → Adjust the Layer 3 prompt. Add emotional stage directions for the problematic section. Regenerate.

---

### Layer 4 Handoff: "Is This ME?"

| # | Criterion | Pass? | If No — What to Fix |
|---|---|---|---|
| 1 | The script sounds like me across the ENTIRE arc, not just in isolated lines | ☐ | |
| 2 | If I read this to a friend, they'd say "yeah, that sounds like you" | ☐ | |
| 3 | All pacing tokens are preserved in their exact positions | ☐ | |
| 4 | All structural beats and argument logic are intact | ☐ | |
| 5 | No banned phrases slipped in | ☐ | |
| 6 | The humor (if any) matches MY style, not generic "internet humor" | ☐ | |
| 7 | Sentence rhythms match my natural speech patterns | ☐ | |
| 8 | The CTA sounds like an invitation from me, not a corporate command | ☐ | |
| 9 | I feel proud to perform this. No cringe. No "this doesn't sound like me" moments. | ☐ | |

**All 9 pass?** ☐ Yes → Script is recording-ready. ☐ No → Identify the specific lines/sections that feel off. Run a targeted revoice prompt for those sections only (see Tool 6: Loop-Back).
