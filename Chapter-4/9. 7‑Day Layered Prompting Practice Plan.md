## 9. 7‑Day Layered Prompting Practice Plan

**Day 1 – Setup**  
- [ ] Copy the Layered Prompting Workspace Template and fill it in for a real topic.  
- [ ] Run Layer 1 (Ideation). Practice using the handoff checklist.  
- [ ] Note any confusion about angle or emotional outcome; clarify your own thinking.

**Day 2 – Skeleton**  
- [ ] Run Layer 2 on yesterday’s beat sheet.  
- [ ] Read the skeleton aloud with no expression. Does the logic hold?  
- [ ] If not, use the Loop‑Back Guide to fix it.

**Day 3 – Pacing**  
- [ ] Run Layer 3 on the approved skeleton.  
- [ ] Read the output aloud twice—once as written, once with actual pauses.  
- [ ] Adjust token placement if any section feels rushed or sluggish.

**Day 4 – Voice**  
- [ ] Run Layer 4 on the paced script using your current voice description.  
- [ ] Read aloud. Highlight any lines that don’t sound like you.  
- [ ] Refine your voice description with more concrete markers.

**Day 5 – Full Run**  
- [ ] Execute all four layers on a brand‑new topic from start to finish.  
- [ ] Time yourself. Aim for under 60 minutes total.  
- [ ] Save the workspace as a permanent template for future scripts.

**Day 6 – Troubleshoot**  
- [ ] Review your last full run. Were any handoff checks skipped?  
- [ ] If a layer produced weak output, revisit the troubleshooting guide and adjust the prompt.  
- [ ] Re‑run that layer only and then propagate forward.

**Day 7 – Reflect and Share**  
- [ ] Compare your layered script to an old one‑shot script. Note differences in voice, pacing, and editing time.  
- [ ] Record a voice memo explaining the four layers as if teaching a fellow creator.  
- [ ] Commit: “From now on, all my scripts are engineered in layers.”
