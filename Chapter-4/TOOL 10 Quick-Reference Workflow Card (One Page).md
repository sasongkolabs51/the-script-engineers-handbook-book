```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   LAYERED PROMPTING — THE FOUR-LAYER CHAIN                      │
│                                                                 │
│   ═══════════════════════════════════════════════════════════    │
│                                                                 │
│   LAYER 1: IDEATION (5–8 min)                                  │
│   → What am I saying? Through what lens?                        │
│   → Output: Beat sheet (5–7 beats, no prose)                   │
│   → Gate: "Would I be excited to record this?"                 │
│                                                                 │
│   ═══════════════════════════════════════════════════════════    │
│                                                                 │
│   LAYER 2: STRUCTURAL SKELETON (8–12 min)                      │
│   → Build the argument in raw, bland prose                      │
│   → Output: Logical skeleton with [HOOK] [CTA] placeholders    │
│   → Gate: "Does it hold together in monotone?"                 │
│                                                                 │
│   ═══════════════════════════════════════════════════════════    │
│                                                                 │
│   LAYER 3: PACING INJECTION (5–8 min)                          │
│   → Add rhythm, tokens, B-roll, energy shifts                   │
│   → Output: Performable script with delivery markers            │
│   → Gate: "Does it breathe when I read aloud?"                 │
│                                                                 │
│   ═══════════════════════════════════════════════════════════    │
│                                                                 │
│   LAYER 4: VOICE OVERLAY (5–8 min)                             │
│   → Apply MY voice to the paced skeleton                        │
│   → Output: Recording-ready script that sounds like ME          │
│   → Gate: "Would my subscribers recognize this?"               │
│                                                                 │
│   ═══════════════════════════════════════════════════════════    │
│                                                                 │
│   THE RULES:                                                    │
│                                                                 │
│   • One layer at a time. Never combine.                         │
│   • Validate each layer before proceeding.                      │
│   • Loop back to the broken layer. Never nuke the chain.       │
│   • Read Layers 3 and 4 ALOUD. Always.                         │
│   • The AI is the Engineer. I am the Visionary + Craftsman.    │
│                                                                 │
│   TOTAL TARGET TIME: Under 60 minutes.                         │
│   TOTAL TARGET EDITING: Under 5 minutes.                       │
│                                                                 │
│   ─────────────────────────────────────────────────────────     │
│   My mantra: "I don't write scripts. I engineer them.          │
│   Layer by layer. Beat by beat. On purpose."                   │
│   ─────────────────────────────────────────────────────────     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
