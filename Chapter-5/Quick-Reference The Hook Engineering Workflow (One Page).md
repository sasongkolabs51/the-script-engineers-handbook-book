## Quick-Reference: The Hook Engineering Workflow (One Page)

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   HOOK ENGINEERING — THE 5-STEP WORKFLOW                        │
│                                                                 │
│   1. SELECT (30 sec)                                           │
│      → Match archetype to emotional goal (Tool 1)              │
│      → Choose retention mechanism                              │
│                                                                 │
│   2. GENERATE (3 min)                                          │
│      → Run Hook Generation Machine (Tool 6)                    │
│      → Get 3 variations                                        │
│                                                                 │
│   3. EVALUATE (2 min)                                          │
│      → Score each with Scroll-Stop Scorecard (Tool 4)          │
│      → Must score 4+ / 5 to proceed                           │
│      → Read the winner ALOUD                                   │
│                                                                 │
│   4. INTEGRATE (1 min)                                         │
│      → Paste winner into Layer 2 prompt (Tool 8)               │
│      → Confirm retention chain carries through skeleton        │
│                                                                 │
│   5. VALIDATE (30 sec)                                         │
│      → After full script: re-read the first 15 seconds         │
│      → Does the hook's energy survive to the body?             │
│      → Is the open loop still open? Stakes still escalating?   │
│                                                                 │
│   ─────────────────────────────────────────────────────────     │
│   THE RULE: Never prompt "make it engaging."                   │
│   Specify the MECHANISM. The AI executes. You validate.        │
│   ─────────────────────────────────────────────────────────     │
│                                                                 │
│   SIX ARCHETYPES:                                              │
│   1. Curiosity Gap    4. Specific Promise                      │
│   2. Contrarian Claim 5. Identity Challenge                    │
│   3. Relatable Pain   6. Open Loop                             │
│                                                                 │
│   THREE RETENTION MECHANISMS:                                  │
│   • Open Loop Extension (deepen the mystery)                   │
│   • Pattern Interrupt (break the expectation)                  │
│   • Stake Escalation (raise from personal to universal)        │
│                                                                 │
│   THE SCORECARD: Specificity / Emotion / Gap /                 │
│   Authenticity / Chain — must score 4+ to ship.               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
