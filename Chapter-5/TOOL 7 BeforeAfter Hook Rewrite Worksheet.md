### Practice Set: Rewrite These (Starter Examples)

Try these before using your own scripts:

| Weak Hook | What's Wrong | Your Rewrite |
|---|---|---|
| "Hey guys, today we're talking about email marketing." | Topic sentence. No hook. No emotion. No specificity. | |
| "Have you ever struggled with time management?" | Generic rhetorical question. Could be any video. No detail. | |
| "In this video, I'll share 5 tips for better sleep." | Summary. No curiosity gap. No stakes. No reason to stay. | |
| "Social media is bad for your mental health." | Generic claim. No specificity. No personal angle. No retention chain. | |
| "I want to tell you about my journey with investing." | Preamble. No interrupt. No tension. Viewer scrolls. | |
