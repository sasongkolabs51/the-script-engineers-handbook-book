## 9. 7‑Day Hook Engineering Practice Plan

**Day 1 – Learn the Archetypes**  
- [ ] Study the Six Hook Archetypes Reference Card.  
- [ ] For each, find one example from a YouTuber in your niche (or a related one) and write down why it fits.  
- [ ] Note which archetype feels most natural for your voice.

**Day 2 – Worksheet and Selection**  
- [ ] Use the Hook Archetype Selection Worksheet on three upcoming video ideas.  
- [ ] Notice patterns: are you gravitating toward one archetype? Is that a fit or a crutch?

**Day 3 – First Hook Generation**  
- [ ] Run the Hook Generation Machine prompt for one video.  
- [ ] Score the three outputs. Pick the best and debug if needed.  
- [ ] Paste the winning hook into your Workspace.

**Day 4 – Full Chain Practice**  
- [ ] Use the Hook‑to‑Skeleton Integration prompt with that hook and a beat sheet.  
- [ ] Run Layers 3 and 4 on the resulting skeleton.  
- [ ] Record a voice test of the final opening and listen for authenticity and rhythm.

**Day 5 – Vary the Mechanisms**  
- [ ] Take the same topic and generate hooks using a different retention mechanism (e.g., if you used Open Loop Extension, try Pattern Interrupt).  
- [ ] Compare the feel. Which one serves the video’s emotional goal better?

**Day 6 – Scorecard Calibration**  
- [ ] Review your last three generated hooks with the Scroll‑Stop Scorecard.  
- [ ] If any scored low, use the Hook Debugging Prompt to fix them.  
- [ ] Save the before‑and‑after as a case study.

**Day 7 – Integration Check**  
- [ ] Update all your Layer 1 prompt templates to include the hook architecture fields.  
- [ ] Write a short memo: “My default hook archetype for [content type] is ________, because ________.”  
- [ ] Teach the six archetypes to a friend or record a voice note explaining them.
