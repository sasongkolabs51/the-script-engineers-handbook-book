## 3. Session Flow Map Template

**Purpose:** Design the narrative loop from hook to payoff. For each video, explicitly map the hook’s curiosity gap, promise, or open loop to a specific callback in the payoff.

| Video Topic | Hook (copy the opening sentence or core hook) | Hook Mechanism (what is the unanswered question, promise, or gap?) | Payoff Callback (exact sentence or phrase that resolves it) |
|-------------|----------------------------------------------|--------------------------------------------------------------------|------------------------------------------------------------|
|             |                                              |                                                                    |                                                            |
|             |                                              |                                                                    |                                                            |
|             |                                              |                                                                    |                                                            |

**Example:**

| Hook | Mechanism | Payoff Callback |
|------|-----------|-----------------|
| “The most expensive laptop on the market is actually the worst choice for creative professionals—and the reason has nothing to do with specs.” | What is the hidden non‑spec reason? | “So no, the problem isn’t the processor or the price—it’s the thermal throttling that silently steals your render time. Now you know what to look for.” |

**Prompt instruction to add to your payoff prompt:**  
“In the Emotional Resolution Arc or the CTA, directly reference the specific question, promise, or curiosity gap established in the hook. Echo the hook’s language (a phrase or concept) so the viewer subconsciously registers the loop closure. The callback must feel earned, not gimmicky.”
