### The Callback Quality Check

| Check | Pass? |
|---|---|
| The callback references a SPECIFIC element from the hook (a number, a phrase, an image, a question) | ☐ |
| The callback shows TRANSFORMATION (the same element, now changed/resolved) | ☐ |
| The callback feels EARNED, not gimmicky (the viewer had to watch the whole video to get here) | ☐ |
| The callback uses language that ECHOES the hook without repeating it verbatim | ☐ |
| A viewer who skipped to the end would feel they're missing something (the loop demands the full journey) | ☐ |
