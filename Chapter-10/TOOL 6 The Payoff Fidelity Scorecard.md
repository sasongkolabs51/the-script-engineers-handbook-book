### The Read-Aloud Test

Read the ENTIRE ending aloud—from the final structural beat through the Emotional Resolution Arc and CTA. Don't stop.

**Does the ending feel like a CONTINUATION of the script, or a NEW SECTION?**

☐ Continuation — the energy flows naturally. No tonal shift.
☐ New section — there's a "click" where the script ends and the "ask" begins.

**If "new section":** The Emotional Resolution Arc is missing or too short. The CTA is interrupting instead of completing.

---

### The Scorecard

| # | Criterion | Pass? | Notes |
|---|---|---|---|
| 1 | **Emotional Resolution present** — Is there a 20–30 second "exhale" before the CTA? (Landing + Expansion + Forward Gaze) | ☐ | |
| 2 | **Session Flow callback** — Does the ending explicitly reference the hook's specific element? (Number, question, image, promise) | ☐ | |
| 3 | **Loop closure felt** — Does the viewer subconsciously register "ah, that's resolved"? | ☐ | |
| 4 | **CTA trigger active** — Is the ask embedded in a psychological trigger (identity, curiosity, proof, utility, debt)? Not a bare "please subscribe"? | ☐ | |
| 5 | **Voice consistency** — Does the CTA sound EXACTLY like the rest of the script? No "presenter voice"? No formality? | ☐ | |
| 6 | **Pacing appropriate** — Are tokens placed correctly? [Slow down] / [Pause] / [Fade out energy] for the landing? Niche-appropriate energy? | ☐ | |
| 7 | **No obligation language** — Free of "please consider," "if you enjoyed," "don't forget to," "it really helps the channel"? | ☐ | |
| 8 | **Forward-pointing** — Does the ending orient the viewer toward what's NEXT? (Their application, the next video, the ongoing journey) | ☐ | |
| 9 | **Niche-appropriate** — Does the ending match audience expectations for this content type? (Not a vlog ending on a review, or a review ending on a vlog) | ☐ | |
| 10 | **The "friend test"** — If a friend watched this ending, would they feel *rewarded* or *pitched to*? | ☐ | |

### Scoring

| Score | Verdict |
|---|---|
| 9–10 pass | Ship it. This ending builds your community. |
| 7–8 pass | Minor tweaks. Adjust the failing criterion. Re-run the CTA Generator for that element. |
| 5–6 pass | Structural issue. The Emotional Resolution Arc is likely missing or the session flow callback is weak. Rebuild. |
| Below 5 | The ending is a tacked-on CTA. Go back to Tool 2 (Resolution Arc) and Tool 3 (Session Flow). Rebuild from scratch. |
