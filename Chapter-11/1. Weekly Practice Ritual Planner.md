## 1. Weekly Practice Ritual Planner

**Purpose:** Protect an hour each week for deliberate skill refinement, separate from any script you need to publish. This is the practice room where your instincts stay sharp.

**Day & Time I’ll reserve each week:** ____________________

### Session Structure (60 minutes)

- **0–15 min — Structural Isolation Drill**  
  - Choose one framework from your library (PAS, Hero’s Journey, Hook‑Story‑Offer, etc.).  
  - Pick a topic that doesn’t naturally fit that framework (e.g., use Hero’s Journey for a tech comparison).  
  - Prompt the AI to produce a beat sheet or skeleton using only that framework.  
  - Read the output and ask: *Where did the framework hold? Where did it break? What does that teach me about the framework’s essential beats?*  
  - **Today’s drill:** Framework: ________ Topic: ________  
  - **Insight:** ________________________________________________

- **15–45 min — Voice or Pacing Variation Lab**  
  *(Alternate weeks: one week voice, next week pacing.)*  
  - **Voice week:** Take your permanent Voice Profile and intentionally modify one dimension (e.g., increase humor density, shift emotional temperature to a register you rarely use). Generate a short script segment (30–60 seconds). Read it aloud. Does it still sound like you, just in a different mood? Where does it break?  
  - **Pacing week:** Take the same script segment and score it three ways: rapid/punchy, slow/reflective, and a hybrid that shifts mid‑script. Record yourself reading each version and listen back. Notice how the rhythm changes the emotional weight.  
  - **Today’s variation:** ________________________________  
  - **Observation:** ________________________________________________

- **45–60 min — Reflection & Log Update**  
  - Open your Engineering Log (Section 2).  
  - Record one insight from today’s practice.  
  - Note any “that didn’t work as expected” moment.  
  - Write one thing you want to test in a real script this week.  
  - **Log entry prompt:** “Today I learned that…”
