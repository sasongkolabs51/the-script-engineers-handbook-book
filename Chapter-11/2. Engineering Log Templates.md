### Section C: Failure Autopsy

*When a script underperforms, trace the weakness back to a specific layer. No blame—only engineering.*

| Video | Symptom (low retention, low engagement, etc.) | Layer Diagnosis (Hook, Structure, Voice, Pacing, Payoff, Niche) | Root Cause | Fix Applied | Result (after re‑test or next video) |
|-------|----------------------------------------------|---------------------------------------------------------------|------------|-------------|--------------------------------------|
|       |                                              |                                                               |            |             |                                      |
|       |                                              |                                                               |            |             |                                      |

**Quick diagnostic questions:**
- If drop‑off is early → Hook or first 30 seconds likely at fault.
- If drop‑off is mid‑video → Structure or pacing likely.
- If comments say “not your usual style” → Voice or niche mismatch.
- If low subscribe conversion → Payoff (CTA/emotional resolution) likely.
