### The Diagnostic Guide

| If you see... | It might mean... | The response is... |
|---|---|---|
| Efficiency dropping (time increasing) | Prompts getting bloated. Over-engineering. Or a new format that's unfamiliar. | Run a "lean prompt" drill in weekly practice. Strip prompts to essentials. |
| Quality dipping | A specific layer is degrading. Voice drift? Pacing mismatch? | Check Failure Autopsy. Run Reset Protocol on the affected layer. |
| Retention slipping in a specific section | Pacing or structural issue at that timestamp. | Map the drop to a specific beat. Adjust the tempo map or framework for that section. |
| Quality high but retention flat | Hook issue. The script is good but the opening isn't stopping the scroll. | Revisit Hook-Retention Architecture. Test a different archetype. |
| Efficiency high but quality dropping | Rushing. Skipping handoff checks. Not reading aloud. | Re-commit to the handoff criteria. Don't skip the read-aloud. |
| All metrics stable | You've plateaued. Comfortable but not growing. | Set a stretch experiment. New niche. New framework. New format. Disrupt the comfort. |
