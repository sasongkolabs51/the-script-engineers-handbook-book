## 6. Lifetime Rhythm Checklist

A one‑page reference to integrate the habits into your calendar.

### Daily (5 min)
- [ ] After finishing a script, update the Chain Archive in your Engineering Log.  
- [ ] Jot down any “ah‑ha” moment immediately in the Breakthrough Tracker.

### Weekly (60 min)
- [ ] Complete the Weekly Practice Ritual (Section 1).  
- [ ] Review last week’s videos; add any new Failure Autopsy entries if needed.  
- [ ] Set a tiny goal for next week’s practice (e.g., “I’ll test a new hook archetype in the structural drill”).

### Monthly (15 min + 5 min log)
- [ ] Complete the Monthly Skill Audit (Section 4).  
- [ ] Update the Progress Dashboard.  
- [ ] Review the Breakthrough Tracker and note any recurring themes.

### Quarterly (30 min)
- [ ] Re‑read your Creator Manifesto from Chapter 3. Update it if your identity, voice, or values have shifted.  
- [ ] Run the full Voice Calibration Loop (Chapter 7) to check for drift.  
- [ ] Review the Failure Autopsy section: are there any systemic patterns that need a permanent prompt fix?  
- [ ] Set a bold experiment for the next quarter (e.g., “I’ll adapt my system for a completely new content format like live streaming scripts”).
