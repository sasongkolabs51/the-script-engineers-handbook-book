## TOOL 10: The 30-Day Framework Fluency Plan

### *Build mastery across all three frameworks through deliberate, progressive practice.*

---

### Week 1: PAS Mastery (Days 1–7)

| Day | Action | Time | Done? |
|---|---|---|---|
| 1 | Read the PAS Beat Script (Tool 2). Fill it in for a real topic from your calendar. Don't run it yet. Just fill in the brackets. | 15 min | ☐ |
| 2 | Run the full chain: Layer 1 (with PAS beat script) → Layer 2 (with guardrails) → Layer 3 → Layer 4. Time yourself. | 50 min | ☐ |
| 3 | Run the Self-Assessment (Tool 8). Score the output. Identify the weakest beat. Re-prompt it. | 20 min | ☐ |
| 4 | Run PAS on a SECOND topic. Different niche or angle if possible. | 50 min | ☐ |
| 5 | Run PAS on a THIRD topic. This time, compress to 5 beats (skip Beat 4 for a shorter video). Notice how the framework flexes. | 40 min | ☐ |
| 6 | Take a PAS script you wrote BY HAND in the past. Decompose it into beats using Tool 6. Compare your intuitive structure to the formal beat script. Where do they match? Where do they differ? | 25 min | ☐ |
| 7 | **Reflection:** Write 3 sentences. What's the PAS beat that's hardest to prompt well? Which guardrail made the biggest difference? What principle have I confirmed? | 10 min | ☐ |

### Week 3: Hook-Story-Offer + Integration (Days 15–21)

| Day | Action | Time | Done? |
|---|---|---|---|
| 15 | Fill in the HSO Beat Script (Tool 4) for a conversion/recommendation video. | 15 min | ☐ |
| 16 | Run the full chain. Focus on: does the Offer (Beat 4) feel like a natural extension of the Story, not a commercial break? | 50 min | ☐ |
| 17 | Self-Assessment. Focus on the Hook → Story transition. Does the hook's open loop get resolved IN the story? | 20 min | ☐ |

| 20 | Run your decomposed framework through the full chain. Does it execute? Where does it break? Adjust the beat descriptions. | 40 min | ☐ |
| 21 | **Reflection:** I now have ___ frameworks in my executable library. Which is my default? Which do I need more practice with? | 10 min | ☐ |

### Week 4: Speed + Range (Days 22–30)

| Day | Action | Time | Done? |
|---|---|---|---|
| 22–24 | Produce 3 scripts for your actual channel using the framework system. Different frameworks for each. Time yourself. Target: under 50 min per script. | 50 min each | ☐ |
| 25 | Decompose TWO more frameworks using Tool 6. (Suggestions: Before-After-Bridge, AIDA, The Question-Answer-Question loop, or any from your course library.) | 40 min | ☐ |
| 26 | Test one of your new decompositions on a real topic. Full chain. | 50 min | ☐ |
| 27 | Run a "blind test": write a script with NO framework (just "write a script about X"). Compare it to your framework-engineered scripts. Feel the difference. | 40 min | ☐ |
| 28 | Build your personal "Framework Library" document: all your beat scripts, saved and organized. This is your executable storytelling vault. | 20 min | ☐ |
| 29 | Teach the decomposition method to someone. Explain it in your own words. "Here's how you turn any framework into AI-executable beats..." | 20 min | ☐ |
| 30 | **Final evaluation:** Compare your Day 2 script to your Day 24 script. Rate both on narrative satisfaction (1–10). Calculate the improvement. | 15 min | ☐ |

---

### The Ongoing Practice (After Day 30)

> **Every script:** Select framework → fill in beat script → add guardrails → run chain → assess.
>
> **Every new framework encountered:** Decompose it (Tool 6) within 48 hours. Test it once. Add to library if it works.
>
> **Monthly:** Review your Framework Library. Refine beat descriptions based on what's working. Add niche-specific guardrails. Remove frameworks you never use.
>
> **The goal:** 6–8 executable frameworks in your library within 90 days. Each one a precision tool. Each one producing scripts with genuine narrative architecture. Not summaries. Not pamphlets. Stories.
