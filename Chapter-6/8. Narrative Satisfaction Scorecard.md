## 8. Narrative Satisfaction Scorecard

*After reading the final script (post‑Layer 4), rate it on these dimensions. Aim for 8+ on the overall before you record.*

| **Criterion** | **Question** | **Score (1‑10)** |
|---------------|--------------|------------------|
| Beat Identification | Can I clearly identify each beat by name as I read? (Fuzzy boundaries = lower score) | |
| Emotional Arc | Does the emotional arc match the intended framework arc? (e.g., Agitation feels genuinely uncomfortable) | |
| Stake Escalation | Do the stakes clearly increase from beat to beat where expected? | |
| Payoff Closure | Does the final beat explicitly reference the first beat (loop closure)? | |
| Voice Integrity | Does the script sound like me, consistently, across all beats? | |
| **Overall Narrative Satisfaction** | **Does the script feel like a complete, emotionally coherent story?** | |

**If overall < 8:** Identify the lowest‑scoring criterion. Re‑prompt the specific beats that failed, or add the corresponding guardrail (e.g., for weak stakes, add Stake‑Escalation Guard; for fuzzy payoffs, add Payoff‑Reference Guard).
