## 9. 7‑Day Framework Fluency Practice Plan

**Day 1 – Audit Your Current Knowledge**  
- [ ] List all the storytelling frameworks you’ve learned (courses, books, experience).  
- [ ] For each, write down its core beats and emotional arc from memory.  
- [ ] Identify which ones you use most often and which you’ve never used with AI.

**Day 2 – Build Your First Beat Script**  
- [ ] Pick your most‑used framework (probably PAS).  
- [ ] Using the Beat Script Builder Template, write out a full beat script for a hypothetical video in your niche.  
- [ ] Add personal mandatory fields that feel authentic to your style.

**Day 3 – Guardrail Customisation**  
- [ ] Study the five universal guardrails.  
- [ ] Create one custom guardrail that addresses a recurring problem you’ve faced (e.g., “No tech‑spec without a user‑experience translation”).  
- [ ] Save your personalised Guardrail Stack.

**Day 4 – Full Run with PAS**  
- [ ] Execute the Deployment Action Plan for a real video using PAS, your beat script, and guardrails.  
- [ ] Note the Narrative Satisfaction score. If below 8, debug.

**Day 5 – Try Hero’s Journey**  
- [ ] Choose a topic that would benefit from a personal story arc.  
- [ ] Build a Hero’s Journey beat script using the template.  
- [ ] Run Layer 2 on it, paying attention to the Ordeal beat—this is often the hardest to get right. Use the Beat Debugging Prompt if needed.

**Day 6 – Speed‑Run Hook‑Story‑Offer**  
- [ ] Create a short‑form script (or a video opening) using Hook‑Story‑Offer.  
- [ ] Focus on the seamless transition from Story to Insight.  
- [ ] Time yourself: can you go from blank page to approved skeleton in under 20 minutes?

**Day 7 – Library and Integration**  
- [ ] Organise your beat scripts into a personal Framework Library in your Prompt Lab.  
- [ ] Update your Layer 1 and Layer 2 templates to include framework selection and guardrail insertion as permanent steps.  
- [ ] Write one sentence summarizing how Framework Fluency has changed the emotional quality of your AI‑assisted scripts.
