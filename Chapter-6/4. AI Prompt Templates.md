## 4. AI Prompt Templates

### 4.1 Framework Beat Script Generator

Use this if you want the AI to expand your chosen framework into a full, field‑specified beat script before you manually adjust it. It’s a meta‑prompt that helps you build your own library of beat scripts faster.

```
I am creating a YouTube script about [TOPIC]. The emotional goal is [GOAL]. I have chosen the [FRAMEWORK NAME] framework.

Please generate a detailed beat script for this framework. For each beat, specify:
- The emotional job (what the viewer should feel)
- The content requirement (what must be included)
- A mandatory field (a specific element that must appear, e.g., a number, a personal anecdote, a callback)
- A prompt snippet I could use to instruct the AI to generate that beat

Make sure the beats follow a natural escalation and that the final beat references the first. The beat script should be ready to use as a structural prompt for a video script.

Framework details:
[Optional: paste a brief description of the framework if it's not a common one.]
```

After generation, review the beats and adjust the emotional job or mandatory fields to better fit your niche.

### 4.2 Full Skeleton Prompt with Framework + Guardrails

This is your upgraded Layer 2 prompt. It takes a completed beat script (from your own library or generated above) and the guardrail stack to produce a structurally sound, emotionally aware script skeleton.

```
Using the following beat script, write a complete script skeleton in raw, functional prose. 

Beat script:
[PASTE YOUR CHOSEN BEAT SCRIPT WITH ALL FIELDS]

Important: Follow the emotional job and content requirements for each beat exactly. The beats must appear in the given order and each must fulfill its mandatory field.

Additionally, apply these guardrails throughout:
[PASTE YOUR SELECTED GUARDRAILS OR THE FULL STACK]

Do NOT add stylistic flourishes, voice personality, or pacing markers—only clean, logical prose that hits every beat. Include [HOOK] at the beginning (already defined by the beat script) and [CTA] at the end.

Output the full skeleton.
```

### 4.3 Beat Debugging Prompt

When a specific beat in the generated skeleton feels weak or misses its emotional job, use this targeted fix.

```
The following beat in my script skeleton is not working. It's supposed to accomplish [EMOTIONAL JOB], but it [DESCRIBE ISSUE: e.g., feels like a summary, lacks a concrete detail, doesn't escalate stakes].

Original beat text:
[PASTE THE OFFENDING PARAGRAPH]

Required elements for this beat:
- Emotional job: [paste]
- Content requirement: [paste]
- Mandatory field: [paste]

Please rewrite ONLY this beat to fully satisfy all requirements while maintaining the surrounding context. Use concrete sensory details and avoid meta‑commentary.
```
