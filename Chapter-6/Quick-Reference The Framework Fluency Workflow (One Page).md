## Quick-Reference: The Framework Fluency Workflow (One Page)

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   FRAMEWORK FLUENCY — THE WORKFLOW                              │
│                                                                 │
│   1. SELECT the framework (Tool 1)                             │
│      → Match to emotional goal + video length                  │
│                                                                 │
│   2. FILL IN the beat script (Tool 2, 3, 4, or 6)             │
│      → Every beat gets: emotional job + content req +          │
│        mandatory field + prompt snippet                         │
│                                                                 │
│   3. ADD guardrails (Tool 5)                                   │
│      → Minimum 3: Concreteness + Summary Prohibition +         │
│        Payoff-Reference                                         │
│                                                                 │
│   4. RUN the Layered Prompting chain (Tool 7)                  │
│      → Layer 1: beat sheet from framework                      │
│      → Layer 2: skeleton with beat script + guardrails         │
│      → Layer 3: pacing (framework-aware)                       │
│      → Layer 4: voice overlay                                  │
│                                                                 │
│   5. ASSESS (Tool 8)                                           │
│      → Beat integrity + emotional arc + callback + score       │
│      → Must score 7+ / 10 to ship                             │
│                                                                 │
│   ─────────────────────────────────────────────────────────     │
│   THE RULE: Never say "use [framework]."                       │
│   Decompose. Specify beats. Assign emotional jobs.             │
│   The framework name is for YOU. The blueprint is for the AI.  │
│   ─────────────────────────────────────────────────────────     │
│                                                                 │
│   THE FIVE GUARDRAILS:                                         │
│   1. Concreteness (sensory details, no abstractions)           │
│   2. Emotional Spike (name the target emotion per beat)        │
│   3. Summary Prohibition (no meta-commentary, BE the script)   │
│   4. Stake Escalation (each beat raises the temperature)       │
│   5. Payoff Reference (final beat echoes first beat)           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
