### The Read-Aloud Test (do this first, always)

Read the script aloud. Start to finish.

**Does it sound like me within the first 3 sentences?** ☐ Yes ☐ No

**Does it STILL sound like me at the halfway point?** ☐ Yes ☐ No

**Does it sound like me in the CTA?** ☐ Yes ☐ No

*(If any answer is "no," drift has occurred. Use Tool 8 to diagnose.)*

---

### The Dimension Check

| Dimension | Check | Pass? | If No — What's Off |
|---|---|---|---|
| **Sentence Architecture** | Are sentence lengths varied per my spec? Are fragments present? Do I start sentences with my natural starters? | ☐ | |
| **Humor** | Is humor present at my specified density? Is the TYPE right (dry/self-deprecating/etc.)? Is delivery flat, not signaled? | ☐ | |
| **Emotional Temperature** | Is the baseline right? Do shifts match my range? Is there any gushing, preaching, or motivational language that isn't mine? | ☐ | |
| **Colloquial Fingerprint** | Are my signature words present? Are banned words absent? Are contractions used? Does the cultural reference universe match? | ☐ | |
| **Token Preservation** | Are ALL pacing tokens still in place? None removed or relocated? | ☐ | |
| **Structural Integrity** | Is the argument logic unchanged? No content added or removed? | ☐ | |

---

### The "Friend Test"

> If I read this script to a friend who knows my content, would they say "yeah, that sounds like you" without me telling them I wrote it?

☐ Yes, immediately
☐ Probably, with a few "that's a bit off" moments
☐ No, they'd ask "did someone help you write this?"

---

### Score

| Rating | Criteria |
|---|---|
| **5/5** | Sounds like my best day. I'd record this as-is. Zero manual tweaks needed. |
| **4/5** | Sounds like me. 1–2 lines need a word swap. Under 2 minutes of manual work. |
| **3/5** | Mostly me, but a section drifted. 5–10 minutes of manual revoicing needed. |
| **2/5** | Significant drift. Multiple sections sound generic. 15+ minutes of repair. |
| **1/5** | Doesn't sound like me. The profile failed. Back to calibration. |

**My score:** ___/5
