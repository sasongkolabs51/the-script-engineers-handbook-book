## 3. Voice Calibration Loop Action Plan

**Purpose:** Execute the four‑step loop—Extract, Profile, Test, Refine—to turn your extraction worksheets into a battle‑tested Voice Profile. Total time: about 60–90 minutes.

### Step 1: Extract (15 min)
- [ ] Gather three writing samples (scripts you’ve recorded, each at least 3 paragraphs).  
- [ ] For each sample, run the **Voice Pattern Extraction Prompt** (Section 4.1) or fill out the Extraction Worksheet (Section 1) manually.  
- [ ] You’ll end up with three draft specifications.

### Step 2: Profile (20 min)
- [ ] Compare the three drafts. Highlight patterns that appear in all three.  
- [ ] Using the **Voice Profile Builder Template** (Section 2), synthesize the common elements into one unified profile. Include any unique elements that feel essential even if they only appeared once.  
- [ ] Write the profile in clear, prompt‑ready language. Save as “Voice Profile v1.0.”

### Step 3: Test (20 min)
- [ ] Take a Layer 3 script (paced, but not yet voiced) from any recent project.  
- [ ] Use the **Permanent Layer 4 Voice Overlay Prompt** (Section 4.3) with your new profile.  
- [ ] Read the output aloud. Mark any line or section that doesn’t sound like you.

### Step 4: Refine (15 min)
- [ ] For each marked section, diagnose which dimension failed.  
- [ ] Adjust that dimension in your Voice Profile. Be more specific, add a banned word, or tighten an emotional range.  
- [ ] Rerun the test on the same script. Repeat until the entire output feels like something you’d say naturally.  
- [ ] Save the refined version. This is now your working Voice Profile.

**Post‑loop:** Once the profile produces consistently authentic output across two different scripts (e.g., one educational and one personal), lock it as your permanent Layer 4.
