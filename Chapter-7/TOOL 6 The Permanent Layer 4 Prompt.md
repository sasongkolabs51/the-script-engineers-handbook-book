### *Your forever prompt. Paste this every time. No more ad hoc voice instructions. No more "make it sound like me."*

```
═══ LAYER 4: VOICE OVERLAY (PERMANENT) ═══

Rewrite the following script using my Voice-DNA Profile below.

[PASTE YOUR COMPLETE VOICE PROFILE FROM TOOL 4 — ALL FOUR
DIMENSIONS, BANNED PHRASES, AND REFERENCE EXAMPLES]

═══ CONSTRAINTS ═══

- Maintain ALL pacing tokens ([PAUSE 1s], [PAUSE 2s], [FAST
  ENERGY], [SLOW DOWN], [B-ROLL: ___], [REACTION BEAT],
  [WHISPER EMPHASIS], [LAUGH BREAK], [BEAT]) in their EXACT
  positions. Do not remove, relocate, or modify any token.

- Maintain ALL structural beats and argument logic unchanged.
  Do not add, remove, or reorder content.

- Maintain the emotional arc as designed. The voice overlay
  changes the LINGUISTIC SURFACE only — word choice, sentence
  rhythm, tone, humor placement — not the underlying structure
  or emotional trajectory.

- The output should sound like me TALKING, not like a script
  being READ. If a sentence sounds "written," rewrite it to
  sound "spoken."

- If any section feels uncertain or could go either way, flag
  it with [VOICE CHECK: note] so I can make the final call.

- Do NOT add content I haven't written. Do NOT "improve" my
  ideas. Do NOT make it "more engaging." Just make it sound
  like me.

═══ OUTPUT ═══

Return the fully revoiced script with all tokens intact.
After the script, add a brief "Voice Fidelity Note":
- 2–3 sentences confirming which dimensions were applied
- Any [VOICE CHECK] flags with your uncertainty noted
- One sentence: "This script sounds like [your name/voice
  metaphor] because [specific dimension evidence]."
```
