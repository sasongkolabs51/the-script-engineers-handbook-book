### The Extraction Prompt (use once per sample)

```
Analyze the following text for voice patterns. This is MY writing.
I need you to identify the specific, measurable linguistic patterns
that make this voice distinctive. Do NOT describe the content.
Focus ONLY on HOW it's written.

"""
[PASTE ONE WRITING SAMPLE HERE]
"""

Analyze across these four dimensions:

1. SENTENCE ARCHITECTURE
- Count the words in each sentence. List them in order.
  (e.g., "14, 6, 22, 4, 18, 7, 21, 5")
- Identify the pattern: is there a rhythm? Long-short oscillation?
  Consistent length? Bursts of fragments followed by long sentences?
- Note punctuation habits: em dashes, ellipses, exclamation points,
  question marks, parentheses. How often? In what contexts?
- Note sentence starters: what words begin sentences? ("And," "But,"
  "So," "Look," "The thing is," etc.)
- Note fragment usage: how many fragments? What do they do?
  (Emphasis? Pacing? Humor?)

2. HUMOR DENSITY AND TYPE
- Count the total paragraphs. Count paragraphs containing humor
  (jokes, wry observations, playful asides, absurd comparisons).
  Calculate the percentage.
- For each humorous moment, identify the MECHANISM:
  self-deprecation / absurd exaggeration / deadpan understatement /
  cultural reference / wordplay / unexpected comparison / other
- Note the DELIVERY: does the humor announce itself (exclamation
  points, "haha") or land dry (stated flatly, no signal)?
- Note PLACEMENT: where does humor appear? After heavy sections?
  At the start? Randomly?

3. EMOTIONAL TEMPERATURE
- Identify the BASELINE register: what's the default emotional
  tone? (Curious? Skeptical? Warm? Analytical? Weary? Excited?)
- Identify the RANGE: how does the temperature shift across the
  text? Where does it warm up? Where does it cool down?
- Note what the voice NEVER does emotionally: (Never gushes?
  Never gets angry? Never uses motivational language? Never
  performs vulnerability?)
- Note how vulnerability is handled: (Stated plainly? Wrapped
  in humor? Avoided? Shared matter-of-factly?)

4. COLLOQUIAL FINGERPRINT
- List words/phrases that appear 2+ times (discourse markers,
  intensifiers, signature phrases)
- List words that are conspicuously ABSENT (formal vocabulary
  that a "standard" writer might use but this voice avoids)
- Note contraction usage: always? Sometimes? Which ones?
- Note grammatical quirks: passive voice? Sentence fragments as
  full sentences? Starting with conjunctions? Specific preposition
  choices?
- Note the cultural reference universe: what world does this
  voice reference? (Tech? Pop culture? Academia? Sports? Cooking?
  Specific shows, apps, memes, eras?)
- Note address style: how does the voice refer to the audience?
  ("You"? "We"? "Folks"? "People"? Never directly?)

OUTPUT: A structured specification for each dimension, using
concrete, measurable language. Not "casual" but "sentences
alternate between 15–22 words and 4–7 words; fragments appear
in ~30% of sentences; em dashes used for asides 2–3 times per
paragraph."
```
