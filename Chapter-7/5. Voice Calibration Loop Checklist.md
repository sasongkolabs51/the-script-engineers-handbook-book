## 5. Voice Calibration Loop Checklist

*Follow this four‑step process once, then revisit if your voice evolves.*

☐ **Step 1: Extract**  
☐ Gathered three authentic writing samples (at least three paragraphs each).  
☐ Ran the Voice Extraction Prompt on each sample.  
☐ Received three structured voice specifications from the AI.  

☐ **Step 2: Profile**  
☐ Filled in the Four‑Dimension Deconstruction Worksheet.  
☐ Identified patterns that appear across all three samples (your stable voice core).  
☐ Decided whether to include elements that appeared in only one sample.  
☐ Synthesized stable elements into a single one‑page Voice Profile document using the Voice Profile Builder.  
☐ Saved as “Voice Profile v1.0” in my Prompt Lab.  

☐ **Step 3: Test**  
☐ Took a structurally sound but unvoiced AI script (Layer 3 output).  
☐ Ran Layer 4 prompt: “Rewrite the following script using this Voice Profile: [paste entire Voice Profile]. Maintain all pacing tokens and structural beats exactly as written.”  
☐ Read the output aloud. Noted any sections that felt “off.”  

☐ **Step 4: Refine**  
☐ Diagnosed which dimension failed for each “off” section.  
☐ Tightened the relevant dimension in my Voice Profile.  
☐ Reran the test on the same script and compared.  
☐ Repeated until the output felt like something I wrote on my best day.  
☐ Stored the final refined version as my permanent Voice Profile.
