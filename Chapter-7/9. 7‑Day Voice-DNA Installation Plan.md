## 9. 7‑Day Voice-DNA Installation Plan

**Day 1 – Gather and Extract**  
- [ ] Collect three writing samples that represent your authentic voice.  
- [ ] Run the Voice Pattern Extraction Prompt on each, or fill out the Extraction Worksheet manually.  
- [ ] Save the three draft specifications.

**Day 2 – Synthesize Your Profile**  
- [ ] Use the Voice Profile Builder Template and the Synthesis Prompt to create a unified profile.  
- [ ] Write it in clear, prompt‑ready language.  
- [ ] Save as “Voice Profile v1.0.”

**Day 3 – First Test**  
- [ ] Take a Layer 3 script and run the Permanent Layer 4 Voice Overlay Prompt with your new profile.  
- [ ] Read aloud. Mark any lines that sound off.  
- [ ] Use the Voice Debugging Prompt on the worst offender.

**Day 4 – Refine and Retest**  
- [ ] Tighten the weak dimension(s) in your profile based on yesterday’s test.  
- [ ] Rerun the voice overlay on the same script and compare.  
- [ ] Repeat until the output feels consistently like you. Save as v1.1 if needed.

**Day 5 – Cross‑Topic Test**  
- [ ] Test the profile on a script with a different emotional register (e.g., a personal story if yesterday was a tutorial).  
- [ ] Verify the voice holds without drifting. If it wobbles, adjust the Emotional Temperature range.

**Day 6 – Permanent Integration**  
- [ ] Open your Layered Prompting Workspace Template and replace any old voice instructions with the permanent Voice Overlay Prompt containing your profile.  
- [ ] Complete the Voice-DNA Integration Checklist.  
- [ ] Archive old voice prompts so you never accidentally use them again.

**Day 7 – Celebrate and Anchor**  
- [ ] Generate a full script using your entire chain, with Voice Profile as Layer 4. Notice how little manual revoicing you now do.  
- [ ] Read the Daily Voice Identity Anchor aloud.  
- [ ] Write one sentence: “The biggest difference in my scripts now that my Voice Profile is locked in is ________.”
